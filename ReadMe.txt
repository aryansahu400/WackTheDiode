To implement this mini project cum game follow these steps:

1) open the "Wack the diode" file in the Ardunio IDE.

2) Connect the ardunio UNO R3 via a usb cable to your computer, select the right board and the correct port under tools menu.

3) Upload the File to the board.

4) disconnect the board from your computer.

5) Open the "Circuit Diagram" png file, and complete the circuit accordingly(you may use a breadboard).

6) Give the board a 5V supply and you are ready to play the game.

7) Do not forget to read the rules of the games as mentioned in the "Rules!" text file.
